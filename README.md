# Get start RFID with Arduino

## First

+ Arduino UNO v3
+ RFID RC522

## Connect

Pin layout should be as follows:

| RFID | Arduino |
|:-----|--------:|
| 3.3V | 3.3V    |
| RST  | 9       |
| GND  | GND     |
| MISO | 12      |
| MOSI | 11      |
| SCK  | 13      |
| SDA  | 10      |
| IRQ  | -       |

 **IRQ** Not connected
